# FinTech Machine Learning Project

## Overview
This project focuses on predicting high-risk financial behaviour using machine learning techniques. The goal is to support better financial decision-making in a fintech environment.

## Problem
Can we predict if a customer is at risk of defaulting based on historical financial behaviour?

## Dataset
- Credit card default dataset
- Includes demographic, billing, and payment behaviour

## Tools & Technologies
- Python (Pandas, Scikit-learn)
- Jupyter Notebook
- Data preprocessing & feature engineering

## Models Used
- Logistic Regression
- Random Forest

## Evaluation Metrics
- ROC-AUC
- F1-score
- Confusion Matrix

## Key Insights
- Payment history is the strongest predictor
- Credit utilisation significantly impacts risk

## Outcome
Built a predictive model to identify high-risk users and support financial decision-making.
